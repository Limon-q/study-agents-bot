# Study Agents Bot

Telegram-бот с командой из трёх AI-агентов на базе [Groq API](https://console.groq.com).  
Модель: `llama-3.3-70b-versatile`.

## Агенты

| Агент | Специализация |
|---|---|
| 🗓 Ассистент | Задачи, расписание, напоминания |
| 🔍 Ресёрчер | Поиск и анализ информации |
| ✍️ Копирайтер | Тексты, структура презентаций |

## Ключевые функции

- **Персонализация** — команда `/profile` сохраняет описание пользователя, все агенты адаптируют ответы под него
- **История диалога** — бот помнит контекст разговора в рамках сессии (последние 10 пар)
- **Авто-перезапуск** — если бот падает из-за сети, он автоматически перезапускается через 5 секунд

## Команды

| Команда | Действие |
|---|---|
| `/start` | Приветствие и меню агентов |
| `/menu` | Показать меню переключения агентов |
| `/profile` | Создать / обновить персональный профиль |
| `/clear` | Очистить историю диалога |
| `/status` | Текущий агент, история, статус профиля |

## Локальный запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/YOUR_USERNAME/study-agents-bot.git
cd study-agents-bot
```

### 2. Установить зависимости

```bash
pip install -r requirements.txt
```

### 3. Создать `.env` файл

```bash
cp .env.example .env
```

Заполнить `.env`:

```
TELEGRAM_TOKEN=токен_от_BotFather
GROQ_API_KEY=ключ_от_console.groq.com
```

Где взять токены:
- **Telegram** — написать [@BotFather](https://t.me/BotFather), команда `/newbot`
- **Groq** — зарегистрироваться на [console.groq.com](https://console.groq.com) → API Keys

### 4. Запустить

```bash
python main.py
```

## Деплой на bothost.ru через GitHub

### Шаг 1 — Загрузить проект на GitHub

1. Создать новый репозиторий на [github.com](https://github.com/new) (можно приватный)
2. В папке проекта выполнить:

```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/study-agents-bot.git
git push -u origin main
```

### Шаг 2 — Создать бота на bothost.ru

1. Зарегистрироваться на [bothost.ru](https://bothost.ru)
2. Создать нового бота → выбрать тип **Python**
3. Подключить GitHub-репозиторий в настройках бота

### Шаг 3 — Настроить переменные окружения

В панели bothost.ru открыть раздел **Переменные окружения** и добавить:

```
TELEGRAM_TOKEN = ваш_токен
GROQ_API_KEY   = ваш_ключ
```

> `.env` файл **не загружается** на хостинг — токены вводятся только через панель управления.

### Шаг 4 — Указать команду запуска

В настройках бота установить команду запуска:

```
python main.py
```

### Шаг 5 — Запустить

Нажать **Start** в панели bothost.ru. Бот запустится и будет работать непрерывно.

### Обновление бота

При изменениях в коде достаточно сделать `git push` — bothost.ru подтянет изменения автоматически (если настроен auto-deploy) или после нажатия **Redeploy** в панели.
